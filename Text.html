Text Formatting Example

1. <b>

Usage: Use the <b> tag to make text bold.

Example: This text is bold.

2. <strong>

Usage: Use the <strong> tag to make text bold.

Example: This text is bold.

3. <em>

Usage: Use the <em> tag to italicize text.

Example: This text is italicized.

4. <u>

Usage: Use the <u> tag to underline text.

Example: This text is underlined.

5. <strike>

Usage: Use the <strike> tag to strike through text.

Example: This text is struck through.

6. <ins>

Usage: Use the <ins> tag to indicate inserted text.

Example: This text is inserted.

7.<del>

Usage: Use the <del> tag to indicate deleted text.

Example: This text is deleted.

8. <sup>

Usage: Use the <sup> tag to superscript text.

Example: This text is in superscript: X2

9. <sub>

Usage: Use the <sub> tag to subscript text.

Example: This text is in subscript: H2O

10. <code>

Usage: Use the <code> tag to display code snippets or computer output.

Example: This text is in monospace font: print("Hello, world!")

11. <var>

Usage: Use the <var> tag to variable text.

Example: This text is in variable type

12. <cite>

Usage: Use the <cite> tag to indicate a citation or reference.

Example: This text is a citation.
